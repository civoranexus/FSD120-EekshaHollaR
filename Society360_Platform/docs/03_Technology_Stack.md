## Overview

Society360 is developed using a modern full stack JavaScript-based architecture, following the technology standards prescribed by the CivoraX Internship Program. The stack prioritizes scalability, security, maintainability, and real-world production readiness.

---

## Frontend Technologies

### Next.js (React 18)

- Framework for building scalable, production-ready web applications
- Supports server-side rendering (SSR) and static site generation (SSG)
- Enables optimized routing and image handling

**Key Benefits:**
- Fast performance
- SEO-friendly
- Component-based architecture

npx create-next-app@latest
### Tailwind CSS

- Utility-first CSS framework
- Enables consistent, responsive, and clean UI design
- Ensures adherence to Civora Nexus Pvt. Ltd. design guidelines

---

## Backend Technologies

### Node.js

- JavaScript runtime for server-side application logic
- Handles API requests, authentication, and business workflows

---

### Express.js

- Lightweight web framework for building RESTful APIs
- Manages routing, middleware, and request handling

---

## Database Technologies

### PostgreSQL

- Primary relational database for structured society data
- Stores users, units, visitors, maintenance records, and billing data
- Ensures data integrity and reliability

---

### Prisma ORM

- Type-safe database ORM
- Simplifies database access and schema management
- Reduces runtime errors and improves developer productivity

---

## API Architecture

- RESTful API design
- Clear separation between frontend and backend
- Secure endpoints protected by authentication and role checks

---

## Security Technologies

- JWT-based authentication
- Role-Based Access Control (RBAC)
- Secure session handling
- Input validation and error handling
