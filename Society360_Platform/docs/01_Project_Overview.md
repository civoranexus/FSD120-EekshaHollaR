## Project Overview

Society360 – Smart Residential Management System is a full stack software application designed to digitally manage the administrative, communication, financial, and security operations of a modern residential society or housing complex.

The system provides a centralized and secure platform that connects residents, management committees, and security or facility staff, enabling efficient coordination, transparency, and accountability. It replaces manual, paper-based, or fragmented digital processes with structured workflows and role-based access.

The platform supports core residential operations such as user authentication, visitor management, maintenance request handling, billing and payments (simulated), and official community communication, while strictly adhering to Civora Nexus Pvt. Ltd. branding and design standards.

---

## Purpose of the Project

The purpose of Society360 is to:

- Digitize residential society operations
- Improve transparency in billing, maintenance, and administration
- Enhance security through controlled visitor access
- Enable effective communication between residents and management
- Provide a scalable and maintainable full stack system

---

## Target Users

- **Residents / Homeowners**  
  - Raise maintenance requests  
  - Pay society fees  
  - Manage visitors  
  - Receive announcements  

- **Management Committee / Administrators**  
  - Manage users and units  
  - Generate maintenance bills  
  - Post announcements  
  - Monitor financial and operational reports  

- **Security / Facility Staff**  
  - Log visitor entry and exit  
  - Track approved visitors  
  - Update maintenance task status  

---

## Key Features Overview

- Secure user authentication with role-based access control
- Visitor pre-approval and real-time gate logging
- Maintenance and grievance tracking with status updates
- Monthly billing and simulated digital payments
- Official announcements and resident communication
- Administrative reporting and analytics dashboard

## Technology Stack Overview

The Society360 platform is developed using modern, industry-standard technologies as prescribed by the CivoraX Internship Program.

- **Frontend:** Next.js (React 18) with Tailwind CSS
- **Backend:** Node.js with Express.js
- **Database:** PostgreSQL with Prisma ORM
- **API Architecture:** RESTful APIs
- **Authentication:** JWT-based authentication with role-based access control
- **Deployment:** Vercel (Frontend), Cloud-hosted backend
- **Version Control:** Git & GitHub (CivoraX Classroom Repository)

The selected stack ensures scalability, maintainability, security, and real-world production readiness.
